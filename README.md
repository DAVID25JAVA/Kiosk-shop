🧾 Customer Survey Kiosk App

A modern, responsive Survey Kiosk Application built using Next.js (App Router) and Tailwind CSS.
This application allows shop customers to quickly provide feedback through an interactive UI.

🚀 Features
🎯 Welcome Screen with survey introduction
📊 Dynamic Questions System (easily extendable)
⭐ Rating Inputs (1–5 & 1–10 scale)
✍️ Text Feedback Input
🔄 Navigation Support
Next / Previous
Skip questions
📈 Progress Indicator (e.g., 1/5, 2/5)
💾 LocalStorage-based Data Persistence
🆔 Unique Session Tracking per User
✅ Confirmation Dialog before Submission
🎉 Thank You Screen with Auto Reset (5 seconds)
🎨 Modern UI with smooth animations
🛠️ Tech Stack
Frontend: Next.js (App Router)
Styling: Tailwind CSS
State Management: React Hooks
Storage: Browser LocalStorage


▶️ Run Locally
npm install
npm run dev

Open: http://localhost:3000
